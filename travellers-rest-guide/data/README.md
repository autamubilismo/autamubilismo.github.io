# Data

Esta pasta guarda os dados legíveis do guia.

- `travellers-rest-data.json`: versão em JSON para consulta e edição humana.
- `../data.js`: versão usada diretamente pelo site no navegador.

Se quiser editar o conteúdo do guia pelo GitHub, a forma mais segura é atualizar o JSON e depois refletir as mudanças em `data.js`.
